교보손글씨2019 폰트 넣는 곳
============================

이 폴더(public/fonts/)에 교보손글씨2019 폰트 파일을 아래 이름으로 저장하세요:

  - Kyobo2019.ttf      (교보손글씨는 보통 .ttf 파일입니다)
  - Kyobo2019.woff2    (있으면 더 좋음 — 없어도 됨)

다운받은 파일 이름이 다르면 "Kyobo2019.ttf" 로 이름만 바꿔서 넣으면 됩니다.

넣고 나면 앱의 제목(설정 등)이 자동으로 이 손글씨체로 바뀝니다.
파일이 없으면 임시로 Gaegu(구글 손글씨체)로 표시됩니다.

교보손글씨2019 다운로드: 무료 폰트로 배포됩니다(교보문고 홈페이지 등에서 검색).
