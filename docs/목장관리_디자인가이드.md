# 목장 관리 — 디자인 가이드 (양떼 · 크리스천 파스텔 테마)

> 왼쪽 크리스천 패턴 이미지에서 **실제 추출한 색**을 기반으로,
> "냥냥 성장일기" 가이드 구조·폰트·모서리 규칙을 그대로 따르되
> 색상만 이 이미지 기준으로 교체한 버전.
> Plain white 대신 크림 배경 + 세이지/골드/스카이 파스텔로 통일.

---

## 🖋 폰트

| 용도 | 폰트 | 폴백 |
|---|---|---|
| 제목/강조 (font-display) | 교보손글씨2019 (Kyobo2019) — 손글씨체 | → Pretendard → system sans |
| 본문 (font-sans) | 나눔스퀘어 (NanumSquare) | → Pretendard → system sans |

- 궁서/명조(serif) **절대 금지** (한글 깨짐).
- Pretendard는 이미 프로젝트에 있음 → 폴백으로 그대로 활용.
- 교보손글씨2019: 무료 폰트, 파일 받아서 self-host 또는 next/font/local로 로드.
- 나눔스퀘어: 웹폰트로 로드 가능.

## 💻 적용 코드 (그대로 복사해서 Claude Code에 전달)

### 1) `globals.css` — CSS 변수
```css
:root {
  /* base */
  --bg: #FBF6EC;
  --card: #F4ECDC;
  --white: #FFFFFF;
  --text: #2C382F;
  --text-muted: #6E7C6C;
  --border: #E4E1D3;

  /* theme */
  --sage: #90B484;        --sage-deep: #4E6A47;  --sage-soft: #DCE7D3;
  --sky: #9FCDEC;         --sky-deep: #5B93C4;   --sky-soft: #E4F0FA;
  --gold: #F0C86E;        --gold-deep: #C9A24C;  --gold-soft: #FBEDC9;

  /* status (출석 3단계) */
  --present: #90B484;     --present-soft: #DCE7D3;
  --excused: #F0C86E;     --excused-soft: #FBEDC9;
  --unconfirmed: #D9645F; --unconfirmed-soft: #F6D8D6;

  /* accent */
  --coin: #F2B33D;
  --star: #F0C86E;
  --danger: #D9645F;

  /* radius */
  --radius-card: 18px;
  --radius-btn: 11px;
}
body { background: var(--bg); color: var(--text); }
```

### 3) 폰트 로드 (Next.js)
- Pretendard: 이미 설정됨 (폴백).
- 나눔스퀘어 + 교보손글씨2019: 폰트 파일을 `app/fonts/`에 넣고 `next/font/local`로 로드하거나,
  `globals.css`에 `@font-face`로 등록. (파일 준비는 Claude Code에 맡기면 됨.)

## 🎨 디자인 원칙 (plain white 탈피)

1. **페이지 배경은 절대 순백(#FFF) 금지** → 크림 `#FBF6EC` 기본.
2. **카드는 흰색 대신 베이지** `#F4ECDC` 또는 흰색 카드에 크림 배경으로 대비.
3. 화면마다 상단이나 배경에 **화면별 배경색**의 soft 톤을 은은하게 → 출석은 세이지, 일정은 스카이 등.
4. 제목은 **교보손글씨2019(손글씨체)**로 따뜻하게, 본문은 나눔스퀘어로 또렷하게.
5. 모서리는 카드 18px / 버튼 11px로 둥글게 (귀여운 느낌).
6. 출석판의 3단계 색(세이지/골드/코럴)은 **가독성 최우선** — 배경 파스텔 위에서도 상태가 한눈에 구분되게.

## 🖼 이미지/일러스트 가이드 (선택)
- 배경은 크림 `#FBF6EC`로 깔면 앱과 자연스럽게 이어짐.
- 양(sheep) 모티브 + 성배·별·팜리프 등 크리스천 요소를 파스텔로.
- 내보내기: PNG, 안전 여백 좌우 확보(모서리 18px에 안 잘리게).
